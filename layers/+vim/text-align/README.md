## Description

文本对齐，最常见的比如 markdown 中的表格对齐。

## Install

启用 text-align Layer.

## Key Bindings

Key Binding        | Mode   | Description
:---:              | :---:  | :---:
<kbd>SPC a =</kbd> | Normal, Visual | 按 = 对齐
<kbd>SPC a         | </kbd>|Normal, Visual | 按 | 对齐
<kbd>SPC a :</kbd> | Normal, Visual | 按 : 对齐
<kbd>SPC x a</kbd> | Normal, Visual | 使用 vim-easy-align 对齐
